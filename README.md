Este projeto consiste em uma calculadora de Índice de Massa Corporal (IMC) desenvolvida em C#, com o objetivo de praticar conceitos de lógica de programação e manipulação de dados.
A aplicação recebe informações de peso e altura, realiza o cálculo do IMC e retorna ao usuário a classificação correspondente, conforme os parâmetros utilizados na avaliação do índice de massa corporal.
